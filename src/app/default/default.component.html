Default Commands

These are the commands available when no event value is set in the configuration. Additionally, if a command is not overridden by the event settings, the default command will still run. For example, if the event has an about command, the event version will run instead of the default version. If the event does NOT have an about command, then the default version will run.

Commands should be prefixed with rosalia! to run. rosalia!about would call the about command.
