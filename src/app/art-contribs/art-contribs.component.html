Contributing Art for Rosalia

Rosalia's art command returns a random image of Rosalia provided by a generous contributor. If you are interested in providing art, please follow these instructions!

First, upload the image to Rosalia's repository. The file should go in the `./img` folder. Name your file yourname#.ext, where "yourname" is your name or username (no spaces!), # is the number of art pieces you have contributed (so if this is your first, use 1), and "ext" is the file extension.

Then head to the Rosalia's code and find the file located at `./src/utils/commands/artList.ts`. This is where you will add your art to her command. At the bottom of the file, just before the closing ] symbol, add the following code: 


{{ '{' }}
  file_name: "The file name from above.",
  artist: "Your name or username.",
  artist_url: "Your homepage link.",
{{ '}' }},

Be sure to replace the values with the correct information. Once this is done, you should submit a pull request as documented in the contributing instructions.

If you have questions, get stuck, or would rather just send the art directly, you may contact us at our Discord Server and we will be more than happy to assist!
