Contributing Art for Rosalia

Rosalia's art command returns a random image of Rosalia provided by a generous contributor. If you are interested in providing art, please follow these instructions!

Uploading your Image

This step is required.

First, upload the image to the documentation repository. The file should go in the ./src/assets/img/art` folder. Name your file yourname#.ext, where "yourname" is your name or username, # is the number of art pieces you have contributed (so if this is your first, use 1), and "ext" is the file extension.

Displaying the Image on the Site

This step is optional - if you are not comfortable with the code, we can handle this part after your image is submitted.

Next, within the documentation repository, head to the file located at ./src/assets/data/artList.ts. Here you will add the code to display your art on the art page. Add this code at the bottom of the file, just before the closing ] symbol: 

{{ '{' }}
  file_name: "The file name from above.",
  art_name: "The name of your art piece.",
  artist: "Your name or username.",
  artist_url: "Your homepage link.",
{{ '}' }},

Then head to Rosalia's code and find the file located at ./src/data/artList.ts. This is where you will add your art to her command. At the bottom of the file, just before the closing ] symbol, add the following code: 

{{ '{' }}
  file_name: "The file name from above.",
  art_name: "The name of your art piece.",
  artist: "Your name or username.",
  artist_url: "Your homepage link.",
{{ '}' }},

Be sure to replace the values with the correct information. Once this is done, you should submit a pull request as documented in the contributing instructions.

Questions?

If you have questions, get stuck, or would rather just send us the art directly, you may contact us at our Discord Server and we will be more than happy to assist!
